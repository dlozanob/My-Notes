## Le conditionnel

- *Utilisation*
	- Faire une demande polie
	- Exprimer un souhait
	- Donner un conseil
	- Parler de l'imaginaire
	- Formuler des hypothèses
	- Exprimer la probabilité

- Exemple
	- Si les zombies envahissaient l'UN, ils feraient des émeutes plus chaotiques
		- Si + imparfait +conditionnel présent

###  Le conditionnel présent

- *Structure*
	- Radical du futur simple + Terminaison imparfait

- Exemple
	- Ce serait un monde
	- J'aimerais connaître le monde entier
	- Il voudrais parler plusieurs langues
	- Est-ce que tu souhaiterais dévenir écrivain


### Le conditionnel passé

- *Structure*
	- Conditionnel présent + participe passé

- Exemple
	- Il aurait pu être un très bon athlète

## Exprimer de l'obligation

- *Devoir*
	- Je dois acheter un ordinateur
- *Falloir*
	- Il faut acheter une vache
- *Avoir besoin de*
	- Il y a besoin d'acheter une vache
- *Subjonctif*
	- Il faut qu'il achète un ordinateur


## Le subjonctif présent

On l'utilise pour formalité.

- *Applications*
	- Conseils
	- Obligations
	- Souhaits, désires, voluntés
	- Autorisations, refus, interdictions
	- Doutes, possibilités
	- Sentiments

- Exemples
	- Je ne pense pas que tu *sois* le meilleur pour faire ce travail
	- Nous avons peur qu'il ne *vienne* pas à l'examen
	- Je voudrais que tu *fassses* la vaiselle

|            | Terminaison |
| ---------- | ----------- |
| Je         | e           |
| Tu         | es          |
| Il/Elle/On | e           |
| Nous       | ions        |
| Vous       | iez         |
| Ils/Elles  | ent         |

>[!Note]
>Régarde que les terminaison pour les cas *Nous* et *Vous* sont les mêmes que s'utilisent à l'imparfait. Pour les autres cas, on utilise les terminaison du présent


La racine qu'on doit utiliser dépend de la personne.
- $1^{ère}$ Racine -> Troisième personne pluriel (je, tu, il/elle/on, ils/elles)
- $2^{ème} Racine$ -> Premier personne pluriel (nous, vous)

- Exemple
	- Venir
		- Je *vienne* ($1^{ème}$ Racine - Ils viennent)
		- Nous *venions* ($2^{ème}$ Racine - Nous venons)


## Le subjonctif passé

- *Structure*
	- Auxiliare + participe passé

Auxiliares:

|            | Être   | Avoir |
| ---------- | ------ | ----- |
| Je         | sois   | aie   |
| Tu         | sois   | aies  |
| Il/Elle/On | soit   | ait   |
| Nous       | soyons | ayons |
| Vous       | soyez  | ayez  |
| Ils/Elles  | soient | aient |

- Exemples
	- Il est possible que l'utilisation de portable *ait nuit* nos compétences sociales
	- J'ai peur qu'il *soit venu*


## Le gérondif

- *Structure*
	- En + Radical + ant

- Cas
	- *Condition*
		- *En mangeant* moins de viande, vous aiderez à mitiger l'effet de serre
	- *Opposition*
		- *Tout en polluant* la planète, les pays industrialisés promulgent des lois pour protéger l'énvironnement
	- *Simultanéité*
		- Il est possible de réduire les émissions de gaz à effet de serre *en choisisant* des objets de qualité et *en n'achentant pas* plus que ce dont on a besoin
	- *Cause*
		- Je fais des achats *en choisissant* des produits bio
	- *Manière*
		- Je suis en bonne santé *en privilégiant* le vélo

>[!Note]
>Pour obtenir le radical d'un verbe irrégulier on le conjuge en présent avec *nous*.
>Verbs irréguliers en gérondif -> En ayant, en étant, en sachant
